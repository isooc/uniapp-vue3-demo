## 构建步骤

```bash
# 克隆项目
git clone https://github.com/jzfai/vue3-mobile-uniapp.git

# 进入项目目录
cd  vue3-mobile-uniapp

# pnpm address https://pnpm.io/zh/motivation
# 安装依赖(建议用pnpm)
# 你可以使用 "npm -g i pnpm" 去安装pnpm
pnpm i

# 启动h5
pnpm run dev:h5

# 启动 微信
pnpm run dev:mp-weixin
```
